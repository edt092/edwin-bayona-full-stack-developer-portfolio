## 🚀 About The Project

This is my personal portfolio website designed to showcase my skills, experience, and projects as a Full-Stack Web Developer. The design focuses on a modern **Glassmorphism** aesthetic, high performance, and a seamless user experience across all devices.

**Live Demo:** [Insertar tu URL aquí]

## 🛠️ Built With

*   **Core:** HTML5, JavaScript (ES6+)
*   **Styling:** Tailwind CSS (Utility-first framework)
*   **Design System:** Glassmorphism UI, Dark Mode
*   **Animations:** CSS Keyframes & Intersection Observer API
*   **Icons:** Material Symbols & Google Fonts

## ✨ Key Features

*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
*   **Interactive UI:** Smooth scrolling, reveal animations on scroll, and typing effects.
*   **Modern Aesthetics:** Custom "Glass" effects, glowing gradients, and animated background orbs.
*   **Performance:** Lightweight architecture ensuring fast load times (Lighthouse optimized).

## 📂 Sections

1.  **Hero:** Introduction with dynamic typing animation.
2.  **About:** Professional bio and timeline of experience.
3.  **Stack:** Grid layout showcasing technical skills (Next.js, React, AI, etc.).
4.  **Projects:** curated list of work with hover effects and detailed descriptions.
5.  **Contact:** Direct links to GitHub, LinkedIn, and Email.
