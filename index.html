Sobre mí

Código que genera resultados reales

Soy un desarrollador web full-stack freelance, enfocado en construir productos web que resuelven problemas de negocio concretos: desde tiendas e-commerce con editores visuales hasta plataformas con integraciones de IA generativa y visualizaciones 3D.

Mi approach combina arquitectura técnica sólida (Next.js App Router, TypeScript, APIs RESTful) con pensamiento de producto: cada decisión de UX/UI está orientada a conversiones, retención y performance.

Trabajo con negocios y emprendedores que buscan contratar un desarrollador web full-stack para llevar su presencia digital al siguiente nivel: código mantenible, sitios posicionados en Google y flujos de WhatsApp que convierten visitas en clientes.

Next.js / React TypeScript Tailwind CSS Node.js IA Generativa SEO técnico
